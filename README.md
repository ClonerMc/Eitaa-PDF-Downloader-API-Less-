# دانلودر هوشمند اسناد PDF ایتا (نسخه ۵.۰)

این نرم‌افزار یک ابزار دسکتاپ متن‌باز، مبتنی بر معماری ناهمگام و فاقد نیاز به API رسمی است که جهت استخراج، دانلود خودکار و مدیریت فایل‌های PDF از کانال‌ها و گروه‌های پیام‌رسان ایتا طراحی شده است. پروژه حاضر به عنوان پروژه درس طراحی الگوریتم در دانشکده آموزش عالی زرند (دانشگاه شهید باهنر کرمان) توسعه یافته است.

## ویژگی‌های کلیدی
- بررسی خودکار و هوشمند نسخه‌های جدید نرم‌افزار متصل به گیت‌هاب بدون مداخله در کارکرد سیستم
- نمایش بنر هشدار وجود آپدیت جدید در داشبورد عملیات جهت اطلاع‌رسانی به کاربر
- دور زدن سیستم حافظه پنهان (Cache) گیت‌هاب برای بررسی آنی و لحظه‌ای وضعیت نسخه آنلاین
- تفکیک فایل‌های تکراری، درج حجم، تاریخ، زمان و وضعیت عملیات به همراه ایجاد لینک دسترسی سریع در خروجی اکسل
- رابط کاربری بومی‌سازی شده مبتنی بر تِم تاریک و فونت تاهوما

---

## راهنمای نصب و راه‌اندازی پیش‌نیازها

برای اجرای صحیح این نرم‌افزار، ابتدا باید محیط پایتون و کتابخانه‌های هسته گرافیکی و پردازشی را روی سیستم‌عامل خود پیکربندی کنید. مراحل زیر را به ترتیب دنبال کنید:

### گام اول: نصب زبان برنامه نویسی Python
۱. به وب‌سایت رسمی پایتون به آدرس [python.org](https://www.python.org/downloads/) مراجعه کرده و آخرین نسخه پایدار (ترجیحاً نسخه ۳.۱۰ یا بالاتر) را دانلود کنید.
۲. هنگام اجرای فایل نصب، **حتماً** تیک گزینه **Add Python to PATH** را در پایین پنجره فعال کنید تا دستورات پایتون در خط فرمان سیستم شناسایی شوند.
۳. فرآیند نصب را به اتمام برسانید.

### گام دوم: نصب کتابخانه‌های وابسته (Dependencies)
محیط خط فرمان سیستم (CMD یا PowerShell) را باز کرده و دستور زیر را به صورت یکجا کپی و اجرا کنید تا تمام کتابخانه‌های مورد نیاز پلتفرم نصب شوند:

```bash
pip install PyQt6 playwright pandas openpyxl


گام سوم: نصب هسته مرورگر ابزارهای پردازشی
این نرم‌افزار برای شبیه‌سازی دقیق و بدون نقض قوانین از هسته اختصاصی مرورگر استفاده می‌کند. پس از اتمام نصب کتابخانه‌های گام دوم، دستور زیر را در ترمینال اجرا کنید تا مرورگر ایزوله روی سیستم شما مستقر شود:
Bash
playwright install chromium
راهنمای گام‌به‌گام استفاده از نرم‌افزار
۱. به پوشه اصلی پروژه مراجعه کرده و فایل Run_Eitaa.exe (یا فایل launcher.pyw از مسیر data/main/) را اجرا کنید.
۲. در بدو ورود، توافق‌نامه سطح کاربر (EULA) و سلب مسئولیت حقوقی را مطالعه کرده و در صورت موافقت، تیک تایید را بزنید تا دکمه ورود فعال شود.
۳. پس از رد شدن صفحه بارگذاری (Splash Screen)، پنل اصلی نرم‌افزار باز می‌شود.
۴. در تب داشبورد عملیات، روی دکمه «۱. اجرای مرورگر» کلیک کنید. منتظر بمانید تا پنجره مرورگر باز شود.
۵. در مرورگر باز شده، وارد حساب کاربری ایتا خود شوید و به کانال یا گروه مورد نظر بروید.
۶. به محیط نرم‌افزار برگردید و روی دکمه «۲. شروع استخراج» کلیک کنید تا فرآیند خزش، دانلود فایل‌های PDF و مستندسازی فایل اکسل آغاز شود.
۷. در پایان کار، می‌توانید با استفاده از دکمه‌های پایین داشبورد، مستقیماً به پوشه فایل‌های دانلود شده یا فایل گزارش اکسل دسترسی داشته باشید.
