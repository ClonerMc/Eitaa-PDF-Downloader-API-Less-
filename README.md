## 🚀 آپدیت بزرگ نسخه ۶.۰ (نسخه پایدار)
در این بروزرسانی، هسته پردازشی و رابط کاربری برنامه کاملاً بازنویسی شده تا محدودیت‌های نسخه وب ایتا به طور کامل دور زده شود:

* **اسکرول هوشمند ۳ لایه (Smart Scroll):** شبیه‌سازی حرکت فیزیکی غلتک موس و تزریق کدهای جاوا‌اسکریپت به صورت معکوس (از پایین به بالا) برای شکستن قفل Virtual DOM ایتا و بارگذاری قطعی پیام‌های قدیمی.
* **استخراج زنده و در لحظه (On-the-fly):** دانلود فایل‌ها و ثبت در گزارش دقیقاً در لحظه رویت (حذف باگ محدودیت ماندگاری فقط ۸ پیام در حافظه موقت مرورگر).
* **حافظه نشست (Session Memory):** جلوگیری هوشمند از دانلود مجدد فایل‌های تکراری در یک بار اجرای برنامه.
* **نسخه تک‌فایل مستقل (Single EXE):** بسته‌بندی کل هسته پایتون، موتور Playwright و رابط گرافیکی PyQt6 در یک فایل اجرایی تمیز و بدون نیاز به نصب پیش‌نیاز.
* **رابط کاربری و لانچر یکپارچه:** حل مشکل مقیاس‌پذیری ویندوز (DPI Awareness)، سیستم بررسی خودکار پیش‌نیازها، بارگذاری کاملاً واقعی (بدون تایمر ساختگی) و پنل لاگ‌گیری یکپارچه.

## 💻 نحوه استفاده

### روش اول: استفاده از نسخه اجرایی (بدون نیاز به نصب پایتون)
1. به بخش **[Releases]** در همین مخزن مراجعه کنید.
2. فایل `Eitaa_Downloader.exe` را دانلود کرده و اجرا نمایید.

### روش دوم: اجرای سورس‌کد
```bash
# 1. نصب پیش‌نیازها
pip install PyQt6 playwright pandas openpyxl

# 2. نصب درایورهای مرورگر
playwright install chromium

# 3. اجرای برنامه
python data/main/launcher.pyw
